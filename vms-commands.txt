__commands for vms:__

__vagrant global-status__  : Get a list of all your VirtualBox virtual machines by name and UUID from the shell

__VBoxManage list vms__  :Get a list of all your VirtualBox virtual machines by name and UUID from the shell

__VBoxManage list runningvms__  : A similar command that lists only the currently running VMs


__VBoxManage unregistervm --delete "<Name of Machine>"__  : delete a vm


__sudo shutdown -h now__  : shutdown the machine
